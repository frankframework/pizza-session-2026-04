# Nutrition Grid Site

A small static website that renders product cards in a responsive grid.

## Features

- Product image cards
- Hover flip animation to show nutrition facts
- Click any card to open a modal with full nutrition details
- Keyboard support (`Enter`/`Space` to open, `Escape` to close)

## Quick start

1. Run tests:

   ```bash
   npm test
   ```

2. Start a local server:

   ```bash
   npm start
   ```

3. Open in your browser:

   `http://localhost:5173`

## Files

- `index.html`: page structure
- `styles.css`: layout, card flip effect, and modal styles
- `data.js`: provided product dataset
- `app.js`: render logic and interaction handlers
- `tests/smoke.mjs`: tiny dataset smoke test
